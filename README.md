# DeepSeek Prompt Quicker

一个专为 DeepSeek 聊天界面设计的提示词管理 Chrome 扩展，让您的 AI 对话更高效、更精准。

## ✨ 功能特性

### 🎯 核心功能

- **内置提示词库**：提供编程、写作、创意等多种场景的预设提示词
- **自定义分类**：支持创建个性化分类，管理专属提示词库
- **智能搜索**：快速查找所需提示词，提高使用效率
- **一键插入**：点击提示词卡片即可插入到输入框，无需手动复制粘贴
- **AI 生成提示词**：使用 AI 根据您的需求自动生成专业提示词
- **模型配置管理**：支持配置多个 AI 模型，灵活切换使用
- **暗色模式支持**：自动适配 DeepSeek 的暗色主题

### 📦 内置提示词分类

- **💻 编程辅助**：代码解释、代码优化、单元测试、Bug 查找、代码转换
- **📝 写作助手**：文章润色、周报生成、内容扩写、标题生成
- **💡 创意生成**：头脑风暴、故事创作、Slogan 设计

## 🚀 安装方法

### 从源码安装

1. 克隆或下载本仓库到本地
2. 打开 Chrome 浏览器，进入扩展程序管理页面（`chrome://extensions/`）
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目的根目录
6. 安装完成！

### 使用要求

- Chrome 浏览器（支持 Manifest V3）
- 访问 `https://chat.deepseek.com/*` 的权限

## 📖 使用指南

### 基本使用

1. **打开提示词面板**
   - 访问 DeepSeek 聊天页面（`https://chat.deepseek.com/`）
   - 在输入框上方找到"提示词"按钮（位于"联网搜索"按钮旁边）
   - 点击按钮打开提示词管理面板

2. **使用提示词**
   - 在左侧分类列表中选择分类
   - 浏览提示词卡片，悬停可查看完整内容
   - 点击提示词卡片即可插入到输入框
   - 使用搜索框快速查找提示词

3. **编辑提示词**
   - 悬停在提示词卡片上，点击"编辑"按钮
   - 修改标题和内容后保存

4. **删除提示词**
   - 悬停在提示词卡片上，点击"删除"按钮
   - 确认删除操作

### 自定义分类

1. 点击"新建分类"按钮
2. 输入分类名称
3. 选择分类图标（从预设图标中选择）
4. 保存后即可在该分类下添加提示词

### 添加自定义提示词

1. 选择目标分类（可以是自定义分类）
2. 点击"新建提示词"按钮
3. 输入提示词标题和内容
4. 保存后即可使用

### 模型配置

1. 在提示词面板中点击"模型配置"按钮
2. 点击"新建模型"添加新模型
3. 配置模型信息：
   - **供应商**：选择 DeepSeek 或火山引擎
   - **Base URL**：API 基础地址（如 `https://api.deepseek.com`）
   - **API Key**：您的 API 密钥
   - **模型名称**：模型标识符（如 `deepseek-chat`）
   - **显示名称**：模型别名（便于识别）
4. 点击"测试连接"验证配置是否正确
5. 设置默认模型以便快速使用

### AI 生成提示词

1. 在提示词面板中点击"AI 生成"按钮
2. 输入您的需求描述
3. 点击"生成"按钮
4. AI 将根据您的需求生成提示词标题和内容
5. 选择分类并保存生成的提示词

## ⚙️ 配置说明

### 权限说明

- **storage**：用于保存自定义分类、提示词和模型配置
- **host_permissions**：用于访问 DeepSeek API 和火山引擎 API

### 数据存储

所有自定义数据（分类、提示词、模型配置）均存储在浏览器本地，不会上传到任何服务器。

## 🛠️ 开发说明

### 项目结构

```
DeepSeekPrompts/
├── manifest.json      # 扩展配置文件
├── content.js         # 主要功能脚本
└── README.md          # 项目说明文档
```

### 技术栈

- **Manifest V3**：使用最新的 Chrome 扩展规范
- **原生 JavaScript**：无依赖，轻量高效
- **Chrome Storage API**：本地数据存储

### 自定义开发

如需修改或扩展功能：

1. 修改 `content.js` 中的 `PREMADE_DATA` 可添加预设提示词
2. 修改样式部分可自定义界面外观
3. 扩展 `PromptManager` 对象可添加新功能
